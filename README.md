TRAVAIL REALISE ET COMMANDES
On a commencé par clner en local le dépôt GitHub pour récupérer le projet.
Avant chaque modification, on utilisait la commande git pull pour récupérer les dernières versions du projet.
Après les modifications, on enregistrait les changements et on les envoyait à GitHub, avec les commandes: git add, git commit et git push.
On a fini par créer un conflit en modifiant la même ligne, que nosu avons ensuite pu résoudre.
